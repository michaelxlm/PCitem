js遍历数组最优方法
for(j = 0,len=arr.length; j < len; j++) { 使用临时变量，将长度缓存起来，避免重复获取数组长度，当数组较大时优化效果才会比较明显。 这种方法基本上是所有循环遍历方法中性能最高的一种 }
